3D Web Developer Portfolio Theme

This theme, 3DWebDevPortfolio_v17, is designed for 3D web developers to showcase their portfolio in an immersive and interactive way. It is fully compatible with Odoo Version 17 CE and incorporates modern design elements, interactive transitions, and a user-friendly layout.

Features

  • Responsive design optimized for desktops, tablets, and smartphones.
  • Interactive elements like scroll animations, hover effects, and smooth transitions.
  • Grid layout for projects with filtering options.
  • Customizable color scheme and typography to match the developer's branding.
  • Professional profile section with skills list and biography.
  • Contact form with social media links.
